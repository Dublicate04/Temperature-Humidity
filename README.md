# 🌡️ IoT Temperature & Humidity Monitor

Real-time temperature and humidity
monitoring using ESP32 and
ThingSpeak cloud dashboard!

## 🛠️ Components
- ESP32 Dev Board
- DHT11 Sensor
- Jumper wires

## 🔧 Features
- Live temp & humidity
- ThingSpeak cloud logging
- Real-time graphs online
- WiFi connected

## 📱 Libraries Used
- DHT sensor library
- ThingSpeak
